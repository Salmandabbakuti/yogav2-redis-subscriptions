# yogav2-redis-subscriptions

Nodejs GraphQL Boilerplate

#### Steps:

1. Install dependencies and run app

```
npm i
npm run linter # checks for linter errors
npm run dev
```

2. Navigate to `http://localhost:4000` and follow API Docs
